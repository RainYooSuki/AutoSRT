# AutoSRT WebUI

这是一个基于Gradio的AutoSRT Web界面，允许您通过网页浏览器轻松使用AutoSRT功能。

## 功能特点

- 直观的Web界面，无需命令行操作
- 支持多文件同时上传处理
- 实时处理进度显示
- 结果预览和下载
- 支持多种音频格式（MP3, WAV, M4A, FLAC, AAC, OGG, WMA, OPUS）

## 启动WebUI

```bash
python webui.py
```

启动后，您可以在浏览器中访问 `http://127.0.0.1:7860` 来使用界面。

## 界面说明

1. **上传音频文件**：点击"Upload"按钮选择一个或多个音频文件
2. **设置输出目录**：指定生成的SRT字幕文件保存的位置
3. **开始处理**：点击"开始处理"按钮开始转换
4. **查看结果**：处理完成后，在下方查看结果和SRT内容预览

## 配置

WebUI会自动加载 `config/config.yaml` 文件中的配置，包括：
- 模型路径
- 计算类型设置
- 音频格式支持
- 模型参数

如需修改配置，请编辑 `config/config.yaml` 文件并重启WebUI。

## 注意事项

- 首次运行可能需要一些时间加载模型
- GPU加速（如果可用）将显著提高处理速度
- 处理大文件可能需要较长时间，请耐心等待
